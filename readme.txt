-- HOW TO USE --
1. Enter the player name
2. Play the game
--- Game Over Conditions - Hit the wall or hit your body HAHAHA
3. Upon clicking, 'OK' in the the game over screen. It saves the data to local storage.
4. Refresh the page to reload the scoreboard and restart the game. Make sure to CTRL + F5 instead of plain F5.

NOTE:
- To clear the DB
1. Inspect Element
2. Click the 'Console' Tab
3. Type localStorage.clear
